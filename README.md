# The Algorithm Portfolio.
A collection of projects designed to highlight my experience with algorithms, data structures, and data analytics.

## Current Projects
### API Earthquake Scraper
The project requests information from an earthquake database to visualize geographical trends for earthquakes in the United States. The purpose of this inclusion is to show familiarity with API webscraping and dataframe manipulation. Data is limited to the last year to remain relevant, and is split into low/high magnitude maps.

### Linear Regression From Fundamentals
A collection of scripts to show the mathematical foundations of linear regression algorithms, focusing on gradient descent and QR matrix factorization. This is a base to be used for a housing project to predict rental costs in Atascadero, CA, with the final goal being to train an LLM using the linear regression data. The predictive model can be used to give a required renter's budget, roommate allocation (number of roommates, bathrooms), if any amenities invoke additional costs, etc.

### Business Requests with OpenAI 
A program that takes in disorganized requests, classifies them, provides next steps, and categorizes based on given variables (priority, risk, problem type, needs human intervention, etc). OpenAI's GPT-4o mini model responds to general queries and its answers are validated to match a given JSON schema. This is my first time working directly with an OpenAI model in a programming project. I wanted to learn the basics of making queries, requesting responses in a specific format, and making sure those answers make sense. The agent is not blindly trusted.
