# The Algorithm Portfolio.
A collection of projects designed to highlight my experience with algorithms, data structures, and data analytics.

## Current Projects
### API Earthquake Scraper
The project requests information from an earthquake database to visualize geographical trends for earthquakes in the United States. The purpose of this inclusion is to show familiarity with API webscraping and dataframe manipulation.

### Linear Regression From Fundamentals
A collection of scripts to show the mathematical foundations of linear regression algorithms, focusing on gradient descent and QR matrix factorization. This is a base to be used for a housing project to predict rental costs in Atascadero, CA, with the final goal being to train an LLM using the linear regression data.
